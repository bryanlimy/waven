**ZebrAnalysis**

packages required:

- python 3.8
- scipy
- scikit learn
- skimage
- tifffile
- pandas
- torch

**installation procedure:**<br />
in conda type :

  conda env create -f environment.yml<br />
  python3 -m pip install --index-url https://test.pypi.org/simple/ --no-deps zebrAnalysis9.54<br />
